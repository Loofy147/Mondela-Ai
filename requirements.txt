# ============================================================================
# SECURITY NOTES
# ============================================================================
#
# Critical Dependencies (must be kept updated):
# - PyNaCl: Cryptographic library, security-critical
# - requests: HTTP client, handles TLS verification
# - psycopg2: Database driver, SQL injection prevention
#
# Security Update Schedule:
# - Run `safety check` before every release
# - Update dependencies monthly (except breaking changes)
# - Monitor GitHub Security Advisories
# - Use Dependabot for automated security updates
#
# Known Security Considerations:
# 1. PyNaCl uses libsodium underneath (vetted crypto library)
# 2. requests uses urllib3 for TLS (verify certificates by default)
# 3. Never install dependencies from untrusted sources
# 4. Always verify checksums when possible
#
# Recommended: Pin exact versions in production
# - Use `pip freeze > requirements-lock.txt`
# - Deploy from locked requirements
# - Test upgrades in staging environment
#
# ============================================================================
# COMPATIBILITY NOTES
# ============================================================================

# Python Version Requirements:
# - Minimum: Python 3.11 (for improved error messages)
# - Recommended: Python 3.12 (for performance)
# - Maximum: Python 3.12 (until dependencies support 3.13)
#
# Operating System Support:
# - Linux: Full support (recommended)
# - macOS: Full support

# Architecture Support:
# - x86_64: Full support

# INSTALLATION TROUBLESHOOTING
# ============================================================================
#
# If `pip install PyNaCl` fails:
#   - Ubuntu/Debian: sudo apt-get install python3-dev libsodium-dev
#   - macOS: brew install libsodium
#   - Windows: Use pre-built wheels (should work automatically)
#
# If `pip install psycopg2-binary` fails:
#   - Ubuntu/Debian: sudo apt-get install libpq-dev
#   - macOS: brew install postgresql
#   - Windows: Use pre-built wheels (should work automatically)
#
# For development on Apple Silicon (M1/M2/M3):
#   - Use native Python 3.11+ (not Rosetta)
#   - All dependencies have ARM64 wheels
